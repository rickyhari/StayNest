# 🏡 StayNest - A Full Stack Property Rental Platform

## 📌 Project Overview

This is a full-stack Airbnb clone built to replicate core functionalities of a property rental platform like Airbnb. The project focuses on backend architecture, authentication, and CRUD operations.

## 🚀 Current Status

🟡 Project is under active development  
✅ Core features implemented  
✅ MVC architecture structured  
🔧 Working on route optimization and additional features  

---

## 🛠️ Tech Stack

* Frontend: HTML, CSS, EJS
* Backend: Node.js, Express.js
* Database: MongoDB (Mongoose)
* Authentication: Passport.js

---

## ✨ Features Implemented

* User Authentication (Register/Login)
* Create, Edit, Delete Listings
* Add and Manage Reviews
* Authorization (Only owner can modify content)
* MVC Architecture for clean structure
